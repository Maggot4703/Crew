Suggestion 14: Hardcoded UI Values

PROBLEM:
Magic numbers and hardcoded values throughout UI code reduce maintainability.

EXAMPLES:
- Default column widths
- Window sizes
- Padding values
- Cache durations

SOLUTION:
Create constants module or configuration:

```python
# UI Constants
class UIConstants:
    DEFAULT_COLUMN_WIDTH = 150
    WINDOW_MIN_WIDTH = 800
    WINDOW_MIN_HEIGHT = 600
    STANDARD_PADDING = 5
    LARGE_PADDING = 10
    
    # File paths
    DEFAULT_DATA_FILE = "./data/npcs.csv"
    LOG_FILE = "crew_gui.log"
    
    # TTS settings
    TTS_CHUNK_SIZE = 500
    CACHE_DURATION = 3600
```

USAGE:
```python
self.data_table.column(col, width=UIConstants.DEFAULT_COLUMN_WIDTH)
```

BENEFITS:
- Easy to modify UI values
- Consistent spacing
- Better maintainability
- Configuration flexibility
